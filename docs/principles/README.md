# Consumer Principle Docs

These docs are a human-curated consumer layer built on top of the focused principle artifact at `data/principles/youtube/kenny-finance-streams/principles_investing_focus.json`.

They are intentionally narrower than the raw artifact.

- They keep the strongest repeated ideas that are useful for downstream readers, bots, or agents.
- They collapse overlapping principles into a smaller set of durable rules.
- They exclude borderline items that matched a focus domain in the artifact but read as lifestyle, stream, or overly mixed-content ideas on review.

Use these docs as the first-pass consumer outputs for the investing-focused Kenny Finance corpus.

## Included Docs

- `public-equity.md`: principles for listed-company analysis and public-market decision making
- `private-equity.md`: principles for business acquisition, diligence, deal process, and operating leverage
- `personal-finance.md`: principles for household capital allocation, savings behavior, leverage caution, and long-term compounding

## Curation Rules

The docs below were written from the ranked principle candidates, but not by blindly preserving rank order.

- Public equity keeps ideas tied to valuation, moat, filings, capital allocation, and market structure.
- Private equity keeps ideas tied to sourcing, diligence, structure, leverage, liquidity, and operational execution.
- Personal finance keeps ideas tied to behavior, liquidity, tax efficiency, debt management, and long-duration compounding.

When one principle fit multiple domains, it was placed where it is most actionable for a downstream consumer.
